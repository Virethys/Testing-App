# MoodBoardZ

Aplikasi web untuk menyimpan dan berbagi inspirasi kreatif dengan fitur mood board.

## Teknologi

- React + TypeScript
- Vite
- Tailwind CSS
- Express.js + MongoDB (Backend)

## Cara Menjalankan

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

## Fitur

- Membuat dan mengelola board
- Upload dan organisasi pin
- Filter berdasarkan mood
- Autentikasi pengguna
